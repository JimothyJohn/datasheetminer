Quick Start

# Install uv package manager
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install dependencies
uv sync

# Set your Gemini API key
export GEMINI_API_KEY="your-api-key"

# Analyze a document
uv run datasheetminer/scraper.py \
  --output device.json \
  --url http://example.com/my.pdf \
  --type drive \
  --pages 1,2,3

Command Options

  • --url - URL of the PDF datasheet to analyze
  • --type - Schema type: motor or drive
  • --pages - Page ranges: 1,2,3 or 1-5 or 1,3-5,7
  • --output - Output JSON file path (optional, defaults to output.json)
